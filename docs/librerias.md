# Librerías

## Librerías incluidas con el core ESP32

El proyecto `001-wifi-scan` usa:

```cpp
#include <WiFi.h>
```

Esta librería se instala junto con el core `esp32 by Espressif Systems`, por lo que no hace falta instalarla aparte desde el Library Manager.

## Librerías externas

Por ahora este repositorio no requiere librerías externas.

Cuando un proyecto necesite instalar una librería adicional, se documentará en el README propio del proyecto e indicará si debe instalarse desde:

```text
Tools → Manage Libraries...
```

o si debe agregarse manualmente.
