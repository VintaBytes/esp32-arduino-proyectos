# Proyectos

Esta carpeta reúne los proyectos prácticos del repositorio.

Cada proyecto está organizado en una subcarpeta independiente, con su propio `README.md` y el sketch listo para abrir desde Arduino IDE 2.

## Criterio de organización

Dentro de cada proyecto, la carpeta del sketch y el archivo `.ino` tienen el mismo nombre, tal como espera Arduino IDE. Por ejemplo:

```text
001-wifi-scan/
└── wifi_scan/
    └── wifi_scan.ino
```
